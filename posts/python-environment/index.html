pipx

pipx replaces a subset of pip’s functionality; it lets you install cli applications but NOT libraries that you import in your code.

Manages the dependencies.

1
2
3
4
5

# global
python -m pip install pipx

# user
python -m pip install --user pipx

Then you can install utils which wont break due to dependencies updating.

1

pipx install visidata

What does it mean to inject

This injects pandas into the visidata virtualenv.

If you have tools with optional dependencies, like visidata, you can inject them into the pipx virtualenv like so:

1
2

#           package  dependencies
pipx inject visidata pandas

pipenv

https://github.com/pypa/pipenv

Automatically creates and manages a virtualenv for your projects, as well as adds/removes packages from your Pipfile as you install/uninstall packages.

It also generates the ever-important Pipfile.lock, which is used to produce deterministic builds.
