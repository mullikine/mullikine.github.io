The difference between lispy's convolute function and paredit's convolute function - 0001-01-01 00:00:00

lispy convolute ;; These two convolution functions are different ;; (special-lispy-convolute) ;; C ;; Swaps parent with grandparent including their elements. Current element holds still. ;; Reversible ;; (paredit-convolute-sexp &optional N) ;; M-P ;; ;; Swaps parent with grandparent including their elements. Current element and siblings after current hold still. ;; Non-reversible (a a1 (b b1 (c c1 (d d1 (e) d9) c9) b9) a9) ;; C on c (b b1 (a a1 (c c1 (d d1 (e) d9) c9) a9) b9) ;; M-P on c (b b1 (a a1 (c c1 (d d1 (e) d9) c9) b9 a9))
