Useful POSIX shell functions: cmd and myeval

Jun 2, 2021
3 min read
shell

Summary

I talk about two shell scripts I frequently use to simplify the handling of arguments, quotations, spaces and escape codes within shell scripts.

They are POSIX-compliant. They are so useful I think they should be builtins.

See also
https://mullikine.github.io/posts/string-quoting-utility/

The string-quoting utility is used to make a script that handles double quotes. That is out of scope of this article.

Demo

1

cmd echo "hello ''''\"simon" thanks for lunch
'echo' 'hello '\'''\'''\'''\''"simon' 'thanks' 'for' 'lunch'

myeval has the inverse action to cmd

1

myeval 'echo' 'hello '\'''\'''\'''\''"simon' 'thanks' 'for' 'lunch'

#!/bin/sh
export TTY

# This also handles unicode correctly

cmd() {
    for var in "$@"
    do
        printf "'%s' " "$(printf %s "$var" | sed "s/'/'\\\\''/g")";
    done | sed 's/ $//'
}

cmd "$@"

#!/bin/bash
export TTY

printf -- "%s" "$(
for (( i = 1; i < $#; i++ )); do
    eval ARG=\${$i}
    if test -z "$ARG"; then
        printf '""'
    else
        # q-eval
        # ARG="$(p "$ARG" | bs "'")"
        ARG="$(printf -- "%s" "$ARG" | sed "s/'/'\\\''/g")"
        echo -n -e "'$ARG' "
    fi
    echo -n -e "$CMD_DELIM"
done
eval ARG=\${$i}
if test -z "$ARG"; then
    # This test is for the specific case of { cmd yo yo yo | cmd - -d "\n"; }
    test "$#" -gt 0 && printf '""'
else
    # q-eval
    # ARG="$(p "$ARG" | bs "'")"
    ARG="$(printf -- "%s" "$ARG" | sed "s/'/'\\\''/g")"
    echo -n -e "'$ARG'"
fi
)"

Usage

Let’s say you wanted to hand around the arguments to a script.

Sure, you can create an array for it. But this is probably easier and more useful.

# Get the script name
sn="$(basename "$0")"

# Capture the arguments to this script as a string
CMD="$(cmd "$@")"

# Print the command invocation to stderr
# You may log this somewhere
echo "$sn $CMD" 1>&2

# Process the arguments
while [ $# -gt 0 ]; do opt="$1"; case "$opt" in
    "") { shift; }; ;;
    -n) {
        window_name="$2"
        shift
        shift
    }
    ;;

    *) break;
esac; done

# The arguments are gone now, but you still have them lying around.
# Run the command on somehost
ssh somehost "$sn $CMD"
