Set up tooling

https://github.com/purescript/documentation

  • pulp (build system for PureScript)
    https://github.com/purescript-contrib/pulp
    https://www.npmjs.com/package/pulp

  • psc-package (Package manager for PureScript)
    Based on the concept of package sets. https://github.com/purescript/psc-package

  • gulp-purescript
    https://github.com/purescript-contrib/gulp-purescript

  • psvm-js
    PureScript Version Manager
    https://github.com/ThomasCrvsr/psvm-js

  • purs-loader (PureScript loader for WebPack)
    https://github.com/ethul/purs-loader/

  • psc-pane (Auto reloading compiler)
    Formats a single error to fit the window https://github.com/anttih/psc-pane

  • purescript-psa (error/warning reporting frontend for psc)
    Pretty, flexible, features colours, original source spans in errors, warning filtering and persistence.
    https://github.com/natefaubion/purescript-psa

  • pscid (file-watcher/testrunner)

    • lightweight
    • like ghcid
    • For PS projects
    • Providing instant-rebuilds in an editor agnostic way

    https://github.com/kRITZCREEK/pscid
