Translating a chord to the help key in terminfo

Mar 30, 2020
2 min read
terminfoemacs
Original article
man terminfo

tmux

tmux is great for translating key bindings. I decided that I’d like M-C-] to be translated into the help key.

However, tmux has no notion of the help key. It can’t be done in tmux.

emacs

Emacs can interpret the chord as the help key.

1

(define-key key-translation-map (kbd "C-M-]") (kbd "<help>"))

However, this only works in emacs. Other programs respond to the help key. It’s a useful key!

terminfo

The capability khlp must be added to the terminfo definition.

1

khlp=\E^]

Test it out

1

TERM=screen-2color emacs -nw -Q

Full screen-2color.ti file

Add this to the terminfo database like so.

1

tic screen-2color.ti
